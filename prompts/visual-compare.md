# Visual Comparison

Compare HTML product before and after changes.

## Instructions
1. Run `bash scripts/screenshot.sh {{FILE_PATH}} /tmp/before.png` on the original version.
2. Apply the changes.
3. Run `bash scripts/screenshot.sh {{FILE_PATH}} /tmp/after.png` on the modified version.
4. Report structural and visual differences:
   - Layout shifts
   - Color/typography changes
   - New or removed sections
   - Math rendering issues
