Q1MEDIUM AP MC 3.9 this Keyword [1]

public class Box {
  private int size;
  public Box(int size) {
    this.size = size;
  }
}

What does the statement this.size = size; do?

  • (A) Assigns the value of the parameter size to the instance variable size.
  • (B) Assigns the parameter size to itself; the instance variable is unaffected.
  • (C) Causes a compile-time error because the parameter and instance variable have the same name.
  • (D) Creates a new local variable named size that is discarded.

Q4HARD AP MC 3.6 Object Reference Param [1]

  • (A) 0
  • (B) 1

System.out.println(b.getSize());

  • (A) 0
  • (B) 10
  • (C) A compile-time error: a constructor cannot invoke another constructor of the same class.

Q9HARD AP MC 3.5 Method Overload Resolution [1]

What is printed (in order, on two lines)?

  • (B) 9.0 then 9.0
  • (C) 9 then 9

Q13HARD AP MC 3.3 Encapsulation Access [1]

What happens at line X?

  • (B) It compiles, but the assignment is silently ignored.
  • (C) A compile-time error: name has private access in Person.
  • (D) A run-time IllegalAccessException is thrown.
Q14HARD AP MC 3.6 Primitive vs Object Params [1]

Consider the following class and code.

public class Wrap {
  public int n;
  public Wrap(int n) { this.n = n; }
}

public static void modify(int x, Wrap w) {
  x   = 999;
  w.n = 999;
}

// in main:
int x = 5;
Wrap w = new Wrap(5);
modify(x, w);
System.out.println(x + " " + w.n);

What is printed?

  • (A) 5 5
  • (B) 999 999
  • (C) 5 999
  • (D) 999 5
