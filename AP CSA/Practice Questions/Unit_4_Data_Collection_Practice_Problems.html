Q7HARD AP MC 4.12 In-place 2D transpose [1]

Consider the following code segment.

public static void transpose(int[][] m) {
  for (int r = 0; r < m.length; r++) {
    for (int c = r + 1; c < m[r].length; c++) {
      int tmp = m[r][c];
      m[r][c] = m[c][r];
      m[c][r] = tmp;
    }
  }
}

// in main:
int[][] g = { {1, 2, 3},
              {4, 5, 6},
              {7, 8, 9} };
transpose(g);
System.out.println(g[0][2] + " " + g[2][0] + " " + g[1][1]);

What is printed?

  • (A) 3 7 5
  • (B) 7 3 5
  • (C) 3 3 5
  • (D) 6 8 5
